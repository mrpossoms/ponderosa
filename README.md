# ponderosa

This is the Ponderosa Fire Protection mono-repo. This repository contains all the code that powers Ponderosa systems, from the publicly facing web-apps to backend services and server configurations.

## Organization

The repo is broken into top-level components. Each subdirectory at the repo root is a component. Each component also contains a README.md which describes it in greater detail. For any component of interest, refer to its readme.
